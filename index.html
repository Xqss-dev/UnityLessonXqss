First Person Controller

Как создать First Person Controller?

Этот скрипт управляет перемещением игрока в Unity.Обязательно смените EDIT>PROJECT SETTINGS>PLAYER>INPUT SYSTEM PACKAGE = Old

using UnityEngine;

public class PlayerController : MonoBehaviour {
    public float speed = 5f; // Скорость передвижения игрока
    public float sensitivity = 2f; // Чувствительность камеры

    void Update() {
        float moveX = Input.GetAxis("Horizontal") * speed * Time.deltaTime; // Передвижение влево/вправо
        float moveZ = Input.GetAxis("Vertical") * speed * Time.deltaTime; // Передвижение вперёд/назад
        transform.Translate(moveX, 0, moveZ);
    }
}
